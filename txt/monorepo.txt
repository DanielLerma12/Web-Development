
--Create monorepo:

-package.json root folder:

{
  "name": "project-name",
  "private": true,
  "workspaces": ["projects-name/*"]
}

-or:

npm install

Create gitignore if not created yet
Remove gitignore, packagejson locked, from inner folders

----------------------------------------------

--Using pnpm instead of npm:

-Install pnpm:

npm install -g pnpm

Remove node modules and package-lock.json:

Remove-Item -Recurse -Force node_modules
Remove-Item package-lock.json

Delete node modules in inner folders

-Create yaml in root folder and config monorepo:

pnpm-workspace.yaml

packages:
  - "projects/*"

-Install pnpm in root folder:

pnpm install

Aprove esbuild, @swc/core if needed:

pnpm approve-builds

-If store folder was generated in D:, ask AI about changing the store folder
-(Deleting all node modules in root and inner folders worked once before changing folders)
